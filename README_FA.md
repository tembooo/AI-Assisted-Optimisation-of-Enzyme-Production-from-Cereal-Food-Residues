# راهنمای سریع فارسی FAIR-SSF-Opt

این پروژه یک پروتوتایپ پژوهشی برای پوزیشن CIRC4FOOD است و از داده واقعی
منتشرشده درباره تولید آنزیم از پسماندهای غلات با تخمیر حالت جامد استفاده
می‌کند.

## ایده اصلی

پروژه دو سؤال را بررسی می‌کند:

1. آیا داده نسبتاً استانداردشده داخل یک مقاله قابل مدل‌سازی است؟
2. آیا مدل روی مقاله‌های ندیده نیز تعمیم پیدا می‌کند؟

برای جلوگیری از نشت داده، در مدل اول خروجی‌های یک آزمایش در یک fold باقی
می‌مانند و در مدل دوم کل مقاله از train کنار گذاشته می‌شود.

## اجرای پروژه در ویندوز

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python scripts/run_pipeline.py
streamlit run app.py
```

## خروجی‌ها

- پروفایل داده و خلاصه cohortها در `data/processed/`
- مدل‌ها در `artifacts/`
- نمودارها در `reports/figures/`
- داشبورد با دستور `streamlit run app.py`

## نکته علمی مهم

ضعف مدل cross-study نتیجه بدی نیست؛ نشان می‌دهد برای AI قابل‌اعتماد در یک
کنسرسیوم باید واحدها، متادیتا، شرایط فرایند، provenance و کیفیت داده به شکل
FAIR و استاندارد ثبت شوند.
